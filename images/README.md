# Images Folder

## Required Image

Add your PDF cover image here:

- **Filename**: `pdf-cover.jpg`
- **Recommended size**: 1200x1600px (3:4 aspect ratio)
- **Format**: JPG or PNG
- **File size**: Keep under 500KB for fast loading

The product page expects the cover image at:
```
images/pdf-cover.jpg
```

You can also use the cover images you shared earlier.
